# Real-Time Chat Application

A full-stack real-time chat application built using the MERN stack.

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB Atlas
- Socket.IO

## Features
- Real-time messaging
- Instant message updates without refresh
- MongoDB cloud database
- Deployed using Vercel and Render
